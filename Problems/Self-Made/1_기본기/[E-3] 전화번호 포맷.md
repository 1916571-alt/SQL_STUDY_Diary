# 전화번호 포맷

> **정보**
> - **날짜**: 2026년 01월 21일
> - **분류**: Self-Made (E-3)
> - **주제**: 문자열 함수
> - **난이도**: ★
> - **재풀이 여부**: X

---

### 문제 설명

전화번호를 XXX-XXXX-XXXX 형식으로 변환하세요. 원본 데이터는 01012345678 형태입니다.

**테이블**: users (id, name, phone)

**출력**: name | formatted_phone

---

### 정답 풀이

```sql
SELECT name,
       CONCAT(
           SUBSTRING(phone, 1, 3), '-',
           SUBSTRING(phone, 4, 4), '-',
           SUBSTRING(phone, 8, 4)
       ) AS formatted_phone
FROM users;
```

---

### 함수 설명

| 함수 | 역할 |
|------|------|
| `SUBSTRING(phone, 1, 3)` | 1번째부터 3글자 → `010` |
| `SUBSTRING(phone, 4, 4)` | 4번째부터 4글자 → `1234` |
| `SUBSTRING(phone, 8, 4)` | 8번째부터 4글자 → `5678` |
| `CONCAT` | `-`로 이어붙이기 |

---

### 배운 점

1. **SUBSTRING 사용법**: `SUBSTRING(문자열, 시작위치, 길이)`
2. **CONCAT으로 조합**: 여러 문자열을 하나로 합칠 때 사용
3. **DATE_FORMAT과 구분**: 날짜는 DATE_FORMAT, 문자열은 SUBSTRING + CONCAT
