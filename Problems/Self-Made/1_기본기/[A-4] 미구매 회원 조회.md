# 미구매 회원 조회

> **정보**
> - **날짜**: 2025년 01월 20일
> - **분류**: Self-Made (A-4)
> - **주제**: LEFT JOIN + IS NULL
> - **난이도**: ★★
> - **재풀이 여부**: X

---

### 문제 설명
2024년 1월에 가입했지만 아직 한 번도 구매하지 않은 회원의
회원ID, 이름, 가입일을 가입일 순으로 출력하세요.

**테이블**: users, orders
**출력**: user_id | name | created_at

---

### 풀이 과정

**핵심 패턴**: LEFT JOIN + IS NULL (미매칭 행 찾기)

```sql
SELECT u.id,
       u.name,
       u.created_at
FROM users u
LEFT JOIN orders o ON u.id = o.user_id
WHERE DATE_FORMAT(u.created_at, '%Y-%m') = '2024-01'
  AND o.user_id IS NULL
ORDER BY u.created_at;
```

---

### 배운 점
- **LEFT JOIN + IS NULL**: 한쪽 테이블에만 있고 다른 테이블에 없는 데이터를 찾을 때 사용
- `o.user_id IS NULL`로 주문 기록이 없는 회원만 필터링
