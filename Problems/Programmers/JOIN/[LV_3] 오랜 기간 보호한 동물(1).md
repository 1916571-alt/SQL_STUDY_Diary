# [LV_3] 오랜 기간 보호한 동물(1)

> **정보**
> - **날짜**: 2026년 1월 20일
> - **분류**: 프로그래머스 (LV_3)
> - **주제**: JOIN

### 문제 핵심
**아직 입양을 못 간 동물 중, 가장 오래 보호소에 있었던 동물 3마리 조회**

1. ANIMAL_INS와 ANIMAL_OUTS를 LEFT JOIN
2. ANIMAL_OUTS에 없는 동물 = 아직 입양 안 간 동물 (`O.ANIMAL_ID IS NULL`)
3. 보호 시작일(DATETIME) 기준 오름차순 정렬 후 LIMIT 3

### 풀이

```sql
SELECT I.NAME,
       I.DATETIME
FROM ANIMAL_INS I
LEFT JOIN ANIMAL_OUTS O
    ON I.ANIMAL_ID = O.ANIMAL_ID
WHERE O.ANIMAL_ID IS NULL
ORDER BY I.DATETIME
LIMIT 3
```
