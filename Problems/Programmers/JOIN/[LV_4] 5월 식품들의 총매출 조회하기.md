# [LV_4] 5월 식품들의 총매출 조회하기

> **정보**
> - **날짜**: 2026년 1월 20일
> - **분류**: 프로그래머스 (LV_4)
> - **주제**: JOIN

### 문제 핵심
**2022년 5월 생산된 식품들의 식품ID, 식품명, 총매출 조회**

1. FOOD_PRODUCT와 FOOD_ORDER를 JOIN
2. 생산일자가 2022년 5월인 데이터 필터링
3. 총매출 = 가격(PRICE) × 주문량(AMOUNT)
4. 총매출 내림차순, 같으면 식품ID 오름차순 정렬

### 풀이

```sql
SELECT fp.PRODUCT_ID AS PRODUCT_ID,
       fp.PRODUCT_NAME AS PRODUCT_NAME,
       SUM(PRICE * AMOUNT) AS TOTAL_SALES
FROM FOOD_ORDER fo
JOIN FOOD_PRODUCT fp
    ON fo.PRODUCT_ID = fp.PRODUCT_ID
WHERE DATE_FORMAT(fo.PRODUCE_DATE, '%Y-%m') = '2022-05'
GROUP BY fp.PRODUCT_ID, fp.PRODUCT_NAME
ORDER BY TOTAL_SALES DESC, fp.PRODUCT_ID
```

### 핵심 패턴
- JOIN + GROUP BY + SUM으로 매출 집계
- 다중 정렬 조건 (내림차순 + 오름차순)
