# 물고기 종류 별 대어 찾기

> **정보**
> - **날짜**: 2025년 12월 28일
> - **분류**: 프로그래머스 (LV_3)
> - **주제**: SUM, MAX, MIN

```sql
WITH
base_info AS(
    SELECT I.ID,
           N.FISH_NAME,
           I.LENGTH
    FROM FISH_INFO I
    JOIN FISH_NAME_INFO N
        ON I.FISH_TYPE = N.FISH_TYPE
),
max_info AS(
    SELECT FISH_NAME, MAX(LENGTH) AS group_max_length
    FROM base_info
    GROUP BY FISH_NAME
)
SELECT bi.ID,
       bi.FISH_NAME,
       bi.LENGTH
FROM base_info AS bi
JOIN max_info AS mi
    ON bi.FISH_NAME = mi.FISH_NAME AND
       bi.LENGTH = mi.group_max_length
```

### 💡 풀이 접근
1. 기본 리스트 테이블 추출
2. MAX_테이블 추출

> [!WARNING]
> **주의사항**
> `bi.LENGTH = mi.group_max_length` 조건만 사용하면 길이만 같으면 JOIN되어 **중복 ROW**가 발생할 수 있습니다. (반드시 물고기 이름도 같이 매칭해야 함)
