# 2022년 10월 16일에 대여중인 자동차 OR 아닌 자동차 찾기

# 1. 렌트 시작일이 2022년 10월 16일 보다 같거나 앞이고 렌트 종료일이 2022년 10월 16일 보다 뒤이면 대여중

#HAVING은 “이 그룹을 결과에 남길지 말지”만 판단할 뿐,그 조건(MAX·COUNT 등)에 해당하는 ‘행’을 골라주지는 않는다.

# 중복 car_id 니까, 조건에 해당하는 것이 1개라도 있는지 찾는 쿼리였다. 그럼 그것을 어떻게 하냐 집계 해버림 대여가능이면 0 대여중이면 1 그래서 group by 별로 값에 따라 대여중이냐 대여가능이냐 나누면 됨.

#그래서 CASE 문 안에 CASE
SELECT
car_id,
CASE
WHEN SUM(
CASE
WHEN START_DATE <= '2022-10-16' AND END_DATE >= '2022-10-16' THEN 1 ELSE 0
END
) > 0
THEN "대여중"
ELSE "대여 가능"
END AS AVAILABILITY
FROM
CAR_RENTAL_COMPANY_RENTAL_HISTORY
GROUP BY
CAR_ID
ORDER BY
CAR_ID DESC
